# AI Hospital Voice Assistant Agent 🏥🎙️

An AI-powered Hospital Management Voice Assistant built using FastAPI, Vapi, and Streamlit with both frontend and backend developed in Python.

This project helps users schedule, book, and manage hospital appointments through real-time voice interaction and an interactive web interface. The system is designed to provide a smooth and intelligent appointment booking experience with fast API handling powered by Uvicorn.

## Features

* AI voice-based appointment booking
* Hospital appointment scheduling system
* Real-time patient interaction
* FastAPI backend for high performance APIs
* Interactive Streamlit frontend
* Vapi integration for voice communication
* Async support with Uvicorn
* Easy to customize and scale

## Tech Stack

* Python
* FastAPI
* Streamlit
* Vapi
* Uvicorn
* Async APIs

A smart and modern AI-driven hospital appointment management solution designed to simplify patient booking and scheduling workflows.
